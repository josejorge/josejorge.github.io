## About

High visual impact single-page developer portfolio built with vanilla HTML5, CSS3, and JavaScript — zero dependencies, zero frameworks, zero build step.

Covers my work across cloud infrastructure, cybersecurity, automation, and game development, plus links to my ecosystems and active projects.

---

## Features

- Animated particle field with connecting node lines (Canvas API)
- Glitch effect on the hero headline
- Custom cursor with lagging ring trail
- Scroll-triggered reveal animations with staggered delays
- Neon glow aesthetic — cyan / magenta / green color system
- Sticky nav that compresses on scroll
- Animated counter stats
- Scanline sweep overlay
- Fully responsive

## Tech

- **HTML5** — semantic structure, single file
- **CSS3** — custom properties, grid, flexbox, keyframe animations, backdrop-filter
- **Vanilla JS** — Canvas API, IntersectionObserver, requestAnimationFrame

No npm. No bundler. No runtime. Open `index.html` and it works.
